# VS Code Flint

[![Visual Studio Marketplace Version](https://img.shields.io/visual-studio-marketplace/v/Flint.flint/label=Visual%20Studio%20Marketplace&logo=visual-studio-code)](https://marketplace.visualstudio.com/items?itemName=Flint.flint)

Provides syntax highlighting and use of the Flint Language Server, which itself provides formatting, goto-definition, autocompletion, and more!

This extension requires the Flint compiler to be installed. If you don't already have the Flint compiler on your system, please follow the instructions at <https://flint-lang.vercel.app/getting-started/> before install this extension.
